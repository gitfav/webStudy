　　和3Q大战时类似，这一次“民意”仍然是周鸿祎发动嘴仗的主要依托，然而此“民意”是否真民意，则依旧没有数据化分析做支撑，仅仅通过转发网友微博的方式就为百度杀毒扣下了流氓软件的帽子，未免显得有些草率了。不过对于“红衣大炮”来说，只要有网友吐槽，其实理由就已经足够了，因为为网友代言一直都是周鸿祎的“专利”，而在市场竞争中首先抢得道德制高点，更是360的常用手法——实话实说，这个桥段真的有些旧，其不说如今的网民早已不像4年前那么好忽悠，就算百度杀毒真的耍流氓，不是也应该呼吁网民通过法律手段来进行维权吗？
