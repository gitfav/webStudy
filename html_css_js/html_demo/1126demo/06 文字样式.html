后盾网 人人做后盾

一场手机专利诉讼大战即将打响!中兴、华为已经向包括小米、OPPO、步步高在内的手机厂商广发“律师函”，理由是侵犯了其WCDMA专利。尽管目前还未上升到诉讼阶段，但小米和OPPO们注定无法逃过专利诉讼这一劫。悲催的是，这次挥起专利大棒的不是国外手机品牌，而是自家兄弟。 与三星与苹果的专利案一样，华为中兴广发律师函，背后还是利益的驱使。小米的高速成长，让华为和中兴颜面扫地。今年10月份，知名分析机构IDC给出的2014年Q3全球智能手机市场报告指出：小米约1800万台出货量成为全球第三大智能手机厂商，占全球市场份额5.3%。与此同时，OPPO于今年7月取代三星，成为国内4G手机出货最多的手机厂商，并持续至今。 与小米的业绩相比，华为和中兴这两位通讯巨头的业绩与其身份严重不符。在今年第三季度，华为手机出货量1650万台，比小米少150万台。中兴的出货量，比华为还要少。显然，小米的快速崛起，让华为和中兴这两大巨头感受到了巨大的压力。
